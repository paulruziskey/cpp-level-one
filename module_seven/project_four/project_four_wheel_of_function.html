Project Four: Wheel of Function

Welcome to Wheel of Function! Your job is to write three functions which each ask a trivia question and return some amount of randomly awarded points if the player’s answer is correct. Zero points are awarded for an incorrect answer. These questions can ask whatever you want, or you can use the same questions shown in the examples below.

Once you have your three functions, you should write a fourth function which randomly returns one of your three trivia functions. This function should then be used whenever the player spins the wheel of function!
